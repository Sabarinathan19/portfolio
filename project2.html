Call Center Data Analysis

Overview

Built a business-centric AI chatbot using Python Flask, MongoDB, and Hugging Face Transformers to streamline support responses for a supply chain platform. This chatbot offers accurate, context-aware replies with real-time history tracking.

Responsibilities

  • Utilized Hugging Face Inference API to power LLM-based Q&A
  • Developed Flask API endpoints for real-time interaction
  • Stored and retrieved chat history using MongoDB for session continuity
  • Deployed with Streamlit frontend, designed to be mobile-friendly
