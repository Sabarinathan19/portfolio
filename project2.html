Call Center Data Analysis

Overview

Analyzed a real-world call center dataset sourced from Kaggle to evaluate performance metrics, identify bottlenecks, and deliver actionable insights for operational improvement. This project simulates a data analyst’s role in optimizing customer support operations.

Tools & Technologies Used

  • Microsoft Excel (Power Query, Pivot Tables, Charts, Formulas)
  • Excel Dashboarding Techniques
  • Data Cleaning & Preprocessing with Power Query
  • KPI-based Reporting

Objectives

  • Understand agent-wise and overall call center efficiency.
  • Track key performance indicators (KPIs) such as:
    • Average Handle Time (AHT)
    • First Call Resolution (FCR)
    • Call Abandonment Rate
    • Customer Satisfaction Score (CSAT)
  • Identify areas for performance improvement.

Project Highlights

  • Used Power Query to clean and transform large volumes of raw data.
  • Built dynamic dashboards to visualize performance across departments, agents, and time periods.
  • Implemented Pivot Tables and advanced Excel formulas (VLOOKUP, FILTER, IF, XLOOKUP) to derive trends and insights.
  • Created KPI cards and charts for high-level executive reporting
  • Generated time-based comparisons (monthly, quarterly) to monitor progress and agent productivity.

Impact & Takeaways

  • Demonstrated the ability to handle real-world business data and communicate insights effectively.
  • Strengthened Excel and data visualization skills relevant for technical and analyst roles.
  • Practiced simulating data-driven decision-making processes for support operations.
