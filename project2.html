Overview

Analyzed a real-world call center dataset sourced from Kaggle to evaluate performance metrics, identify bottlenecks, and deliver actionable insights for operational improvement. This project simulates a data analyst's role in optimizing customer support operations.

Key Performance Indicators

Avg Handle Time
AHT
First Call Resolution
FCR
Abandonment Rate
CAR
Customer Satisfaction
CSAT

Project Highlights

  • Used Power Query to clean and transform large volumes of raw data
  • Built dynamic dashboards to visualize performance across departments and agents
  • Implemented Pivot Tables and advanced Excel formulas (VLOOKUP, FILTER, XLOOKUP)
  • Created KPI cards and charts for executive-level reporting
  • Generated time-based comparisons (monthly, quarterly) to monitor productivity

Impact & Takeaways

  • Demonstrated ability to handle real-world business data at scale
  • Strengthened Excel and data visualization skills for analyst roles
  • Practiced simulating data-driven decision-making for support operations
  • Communicated insights effectively to a non-technical audience

Tools & Technologies
